\usepackage{lipsum}

\begin{document}

\section*{Short quote}

A short quote (usually a line) is done using the environment \emph{quote}:

\begin{quote}
Lorem ipsum dolor sit amet, consectetuer adipiscing elit.
\end{quote}

\section*{Long quote}

A long quote (usually a text) is done using the environment \emph{quotation}:

\begin{quotation}
\lipsum
\end{quotation}
